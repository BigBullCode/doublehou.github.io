use python edit xlsx

2016年12月12日

Yesterday , a girl ask me to solve a problem
that need to change cell’s content in more than
ten thousand regular xlsx files.To show power
to the opposite sex,I say Yes of course.

Then I easily find some python tools working with
xlsx and xlsx:xlrd,xlrt,openpyxl.so I choose openpyxl.

but I noticed that xlsx file’s style be changed by openpyxl
which is unacceptable.
So I ask for help at archlinuxcn bbs.Quckily ,a nice
one find the problem(I wonder it is encode problem at first)
is style , and give his advice that do it on windows and use
com connector.

Then , I esaily find pywin32 ,It can solve my problem!

then,I did it .
push my ugly code and a example file soon.

pywin32 throw a error:can’t find name.xlsx

later , I see the error.
finally,I find a way to solve it :
use os.path.abspath(filename) to get real path then
pass it to pywin32.

wish that is real end .
